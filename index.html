subir un repositorio

  1. // Iniciar un nuevo repositorio
    // Crear la carpeta oculta .git
    git init
  2. // Ver que archivos no han sido registrados
    git status
  3. // Agregar todos los archivos para que esté pendiente de los cambios
    git add .
  4. // Crear commit (fotografía del proyecto en ese momento)
    git commit -m "primer commit"
  5. git log --one line
  6. // Muestra la lista de commit del mas reciente al más antigüo
    git log

or push(para subir a github)

  1. git remote add origin https://github.com/codeAnyeliRodriguez/miprimerproyectogit.git
  2. git branch -M main
  3. git push -u origin main

Trucos

  1. // Muestra en una línea los commit realizados
    git log --oneline
  2. // Muestra en una línea los commit realizados pero más elegante
    git log --oneline --decorate --all --graph
  3. // Solo muestra los archivos modificados
    git status -s

Diferencias entre -- y -

--decorate hace referencia a una palabra -s hace referencia al comando o a varios comandos, -sa serían dos comandos diferentes


// Vemos información de la rama maestra
git status -s -b
git status -sb //Hace lo mismo que el comando anterior
